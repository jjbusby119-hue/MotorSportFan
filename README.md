## All Things Motorsports

A simple, static informational webpage about motorsports, covering Formula racing and endurance racing. Built with HTML and CSS.

## Overview

This page introduces visitors to different types of motorsports, including:

--Formula racing (F1, F2, F1 Academy)

--Endurance racing (WEC, ELMS, NLS)

It includes:

--An image of a Formula 1 car

--Navigation links to jump between sections

--Side notes (aside elements) with extra racing facts

--A footer with author and contact information
